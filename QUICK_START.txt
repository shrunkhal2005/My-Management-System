🆘 TROUBLESHOOTING:

Problem: "Download failed" or "Connection timeout"
├─ Check: Do you have internet connection?
├─ Try: Run again - may be temporary network issue
└─ Alt: Download manually from https://dev.mysql.com/downloads/connector/j/

Problem: "Access denied" or "Connection refused"
├─ Check: Is MySQL Server running? (Windows Services → MySQL Server 9.7)
├─ Try: Start MySQL Server manually
└─ Verify: Credentials are root / shrunkhal@2005

Problem: Still showing "OFFLINE MODE" after download
├─ Check: Did download complete? Look for lib\mysql-connector-java-8.0.33.jar
├─ Try: Manually download from MySQL website
└─ Alt: Try MariaDB driver: Download-MySQLDriver.ps1 -UseMariaDB
